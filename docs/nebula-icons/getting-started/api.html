The API

Here you can reference what calling the library does, and what it returns.

Summary​

Properties​

PropertyTypeDescription
MaterialTableA table containing all the Material Icons from Google as of 2025. It has been discontinued.
LucideTableA table containing all the Lucide Icons from Lucide Team as of 0.539
PhosphorTableA table containing all the Phosphor Icons as of August 2025
Phosphor-FilledTableA table containing all the Phosphor Icons as of August 2025 in the Filled Style Variation
SFTableA table containing all SF Symbols. Imported to Roblox by biggaboy212

Methods​

MethodArgumentsDescription
GetIconname : string, source : stringReturns the rbxassetid of the Icon requested by name within the source icon pack.
