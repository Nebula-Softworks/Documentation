About Elements

Elements are the most basic way to add interactivity to your script. Its the whole reason why we create GUIs and not just simple scripts.
View the sidebar on all the elements and how to create them in your groupbox/tabbox tab.
Here are universal methods that can be used on all base elements.

Setting Element Settings​

ElementVariableName:Set({
    -- New values here.
    -- Unspecified values will use the previous ones
})

Locking and Unlocking Elements​

ElementVariableName:Lock("Reason here, can be left nil to have no reason")

ElementVariableName:Unlock()

Destroying Elements​

ElementVariableName:Destroy()

Accessing Elements Values​

Use ElementVariableName.Values to get returned all the current options/values of the element used on.
To get the Instance, use ElementVariableName.Instance

For the dropdown, please note that it's Instances and it returns a table of the popup and element within the groupbox.

For All Of These, Replace ElementVariableName with the Element you wish to use these on.​

Notes​

For any Element that has a callback (or 2 for Binds), while it is fine to set it within the initial creation parameters,
I highly recommend using :Set({}) to set the callback after creating the Element. It is better to seperate Interface And Functionality.
However, it is all your preference.
